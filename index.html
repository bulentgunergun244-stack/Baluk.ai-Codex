Baluk Engine

BLM kod dili için ilk temel sistem

BLM, Baluk Engine içinde çalışan deneysel bir mini kod dilidir. Matematik işlemleri, yazı yazdırma, değişkenler, koşullar ve öğretici sistemi içerir.

BLM Öğretici

BLM içinde örnek komutlar:

print "Merhaba Dünya"
math 5 + 3
set isim = "Kaan"
print isim
set yas = 13
math yas * 2
if yas > 10 then print "Büyük"
repeat 3 print "Baluk"

Komutlar:

  • print "yazı" → yazıyı ekrana basar
  • math 5 + 3 → matematik işlemi yapar
  • set x = 10 → değişken oluşturur
  • print x → değişken değerini yazdırır
  • if x > 5 then print "tamam" → koşul çalıştırır
  • repeat 3 print "selam" → komutu tekrar eder
  • clear → çıktıyı temizler
  • help → yardım gösterir
